mac sed 报错

2019年5月13日
| 工具命令

[-f script_file] ... [file ...] -i 后边需要添加备份文件的后缀名,如果不需要可以使用"",但是不可以忽略 如 sed -i ".bak" 's/xxxx/yyy/g' test 会将替换后的文本写入test.bak ……
